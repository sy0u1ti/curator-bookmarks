## Store Wording Constraints

- Say "local-first" only as: core bookmark read/search/organize operations run locally by default.
- Say "no default remote behavioral telemetry" only as: Curator does not upload behavior analytics to developer servers by default.
- Do not say "never connects to the internet" because Newtab web search, remote backgrounds, link checks, AI/Jina and user-entered remote media can leave the device.
- Do not say webpage search queries stay local. Only local bookmark search stays local.
- Do not list Featured Gallery as a default third-party request; default background is local color.
- Do not treat the local AI usage ledger as analytics telemetry; it only enforces the user-visible daily AI request limit and is excluded from ordinary backups.
