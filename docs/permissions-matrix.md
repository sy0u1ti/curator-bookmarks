# Curator Permissions Matrix

Last reviewed: 2026-05-08

This matrix is the release-blocking source for Chrome Web Store permission review. Newtab web search and remote backgrounds are documented as external requests, but they are not used as a reason for host permissions.

## Reviewer Test Steps

1. Install the extension, open a new tab, and confirm the first-run strip explains Newtab replacement and restore-help path.
2. Open Options -> Privacy & Permissions Center and verify every permission above is represented.
3. Use Popup and Dashboard search without configuring AI; confirm local search works.
4. In Newtab settings, disable web search and verify local bookmark search remains while web fallback disappears.
5. In Newtab settings, leave background type as Color and verify Featured Gallery is not selected by default; then opt into Featured Gallery and review attribution.
6. In Options -> Availability, run a small fixture check and inspect status/redirect categories; cancel a run to verify task control.
7. Configure AI with a test provider only if available; otherwise verify AI/Jina remain off and local features still work.
